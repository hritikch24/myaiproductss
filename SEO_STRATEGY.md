# SEO Strategy for KraftAI

## Executive Summary
Target: Rank #1 for software development services in India
Primary Keywords: "custom software development", "website development", "mobile app development", "e-commerce development"
Target Audience: Indian businesses, startups, entrepreneurs looking for software solutions

---

## Site Architecture

### URL Structure (Flat & SEO-Friendly)
```
kraftai.in/                          (Homepage - Authority Page)
kraftai.in/services                  (Services Overview)
kraftai.in/services/websites         (Service Category)
kraftai.in/services/websites/[theme] (Service Detail - Lower priority)
kraftai.in/services/stores
kraftai.in/services/mobile-apps
kraftai.in/services/business-apps
kraftai.in/services/design
kraftai.in/services/ai-solutions
kraftai.in/legal-docs                 (Product - Authority)
kraftai.in/padhai                     (Product - Authority)
kraftai.in/enterprise                 (Product)
```

### Keyword Strategy

**Homepage (kraftai.in)**
- Primary: "custom software development company India"
- Secondary: "website development services", "mobile app development India"
- Long-tail: "freelance developer for startup", "build my idea into app"

**Services Pages**
- /services/websites - "custom website development", "responsive website design"
- /services/stores - "e-commerce website development", "online store builder"
- /services/mobile-apps - "mobile app development company", "iOS Android app developer"
- /services/business-apps - "business software development", "custom ERP CRM"
- /services/design - "UI UX design services", "futuristic website design"
- /services/ai-solutions - "AI development company", "machine learning solutions"

## On-Page SEO Checklist

### 1. Meta Tags (Every Page)
- [x] Unique Title (60 chars max)
- [x] Unique Meta Description (155 chars max)
- [x] Canonical URL
- [x] Viewport meta tag
- [x] Robots meta (index, follow)

### 2. Structured Data (JSON-LD)
- [x] Organization schema
- [x] WebSite schema with SearchAction
- [x] LocalBusiness schema
- [x] Service schema for service pages
- [x] Product/SoftwareApplication for products

### 3. Technical SEO
- [x] XML Sitemap (auto-generated)
- [x] Robots.txt
- [x] Open Graph tags
- [x] Twitter Cards
- [x] Favicon
- [x] hreflang (India locale)
- [x] Loading speed optimization

### 4. Content SEO
- [x] H1 (one per page)
- [x] H2-H6 hierarchy
- [x] Image alt text
- [x] Internal linking
- [x] External linking to authoritative sources

### Required Schema Types
1. Organization - Main site
2. WebSite - Homepage
3. Service - Each service category

5. HowTo - Process explanations

- FID: < 100ms
- CLS: < 0.1
